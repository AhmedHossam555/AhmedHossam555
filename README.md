## 💡 Angular Tip of the Day

> **Use `trackBy` with `*ngFor` to prevent unnecessary DOM re-renders.**
>
> ```typescript
> trackByFn(index: number, item: any): number {
>   return item.id;
> }
> ```
> Pairs perfectly with OnPush change detection for blazing-fast list rendering. ⚡
